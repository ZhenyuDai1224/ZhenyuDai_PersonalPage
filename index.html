Leo Rover Automatic Retriever Robot Group Project

In our project to develop an automated object retrieval robot that incorporates SLAM, a manipulator, object detection, and color recognition, I have focused on the mechanical design of the payload sleds while collaborating on the manipulator system and integrating object detection with the manipulator. The video showcases the work done in the project.

UAV Cascade PID Control: Group Feedback Control Project

This is part of a team project for UAV position stabilization, where each team member implements an advanced feedback control algorithm. I implemented a cascaded PID controller with three layers: position (outer), velocity (middle), and acceleration (inner). Additionally, I implemented a separate simple PID controller dedicated to stabilizing the yaw position, ensuring accurate heading control. The overall system outputs velocity and yaw rate commands to guide a quadcopter to desired positions in a simulated 3D environment. The video demonstrates my controller achieving stable and accurate positioning.

LabView project on data acquisition

In a project designed to capture the pattern of an unknown image over a 100mm x 100mm area, we developed a LabView program that utilized an XY plotter paired with an inductive proximity sensor. My primary responsibility was to create the LabView code that accurately navigated the sensor throughout the measurement process.

Mechanical design project on biaxial testing

In a project focused on cost reduction, I spearheaded the design of a pioneering biaxial loading fixture compatible with uniaxial testing machines. My responsibilities included creating a detailed selection matrix to evaluate component options and meticulously integrating various parts into a cohesive assembly using SolidWorks.

A* Path Planning Algorithm Implementation

In this project, I implemented the A* path planning algorithm in Python within a GUI framework. The robot was required to plan a path from a user-defined start to goal location while avoiding user-defined obstacles. The algorithm uses 4-directional motion and the Euclidean distance as the heuristic function. The video demonstrates the successful execution of my implementation in real time.

Review on PID controllers

I conducted a concise literature review on Proportional-Integral-Derivative (PID) controllers, covering their historical origins, the underlying control algorithm, and the distinctions between ideal and standard forms. I also analyzed the common limitations of the PID approach and explored various improvements proposed in the literature. This review helped build a strong foundational understanding of PID control systems and their practical applications in modern control engineering.
